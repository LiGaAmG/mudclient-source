# Local AI — помощник игрока

Модуль встроенного AI-помощника для MUD-клиента Adan. Работает полностью локально, без интернета и GPU.

## Что умеет

- Отвечает на вопросы по текущей зоне и комнате (`/ai <вопрос>`)
- Ищет информацию в локальных файлах лора
- Запоминает комнаты, выходы, мобов и предметы между сессиями
- Строит маршруты по известным комнатам (`/ai route <комната>`)
- Изредка комментирует важные события (можно выключить)
- Сохраняет ручные заметки игрока (`/ai remember <текст>`)

## Установка модели

1. Скачай модель Qwen3 1.7B в формате GGUF (квантизация Q4_K_M):
   - Пример: `qwen3-1.7b-q4_k_m.gguf` (~1.1 GB)
   - Источник: https://huggingface.co/

2. Положи файл в папку `Models\` рядом с клиентом:
   ```
   Adan.Client.exe
   Models\
     qwen3-1.7b-q4_k_m.gguf
   AI\
     Adan.AI.Host.exe
   ```

3. Убедись что в той же папке есть `AI\Adan.AI.Host.exe` (поставляется вместе с клиентом).

## Включение AI

В настройках клиента (Options → Local AI) или командой:
```
/ai on
```

Укажи путь к файлу модели и нажми «Загрузить модель». Первая загрузка занимает 5-30 секунд.

## Команды

| Команда | Описание |
|---------|----------|
| `/ai <вопрос>` | Задать вопрос помощнику |
| `/ai status` | Статус модели |
| `/ai on` / `/ai off` | Включить / выключить AI |
| `/ai cancel` | Отменить текущую генерацию |
| `/ai remember <текст>` | Сохранить заметку |
| `/ai lore reindex` | Переиндексировать папку лора |
| `/ai comments on` / `off` | Включить / выключить фоновые комментарии |

## Папка лора

Положи `.txt`, `.md` или `.json` файлы с описаниями игрового мира в папку `Lore\`:
```
Lore\
  zones\
    dark-forest.md
  mobs\
    trolls.txt
```

После добавления файлов выполни `/ai lore reindex`.

## Где хранится база

`Data\ai-memory.db` — SQLite база с картой комнат, мобами, событиями и индексом лора.

Для сброса памяти: удали `Data\ai-memory.db`.

## Рекомендуемые настройки

**Слабый компьютер (4-8 GB RAM, 2-4 ядра):**
- Threads: 2
- ContextSize: 2048
- MaxResponseTokens: 100

**Средний компьютер (8-16 GB RAM, 4-8 ядер):**
- Threads: 4
- ContextSize: 4096
- MaxResponseTokens: 200

## Известные ограничения

- FTS5-поиск по лору недоступен с System.Data.SQLite из NuGet (лор индексируется, но поиск возвращает пустой результат). Планируется замена на SQLite со статической линковкой FTS5.
- Первый запрос после загрузки модели может занять 10-30 секунд.
- Сайдкар-процесс `Adan.AI.Host.exe` требует .NET 8 Runtime.
- Модель не управляет персонажем — ответы выводятся только локально, не отправляются на сервер.
